<< テニスボールの奇妙なシューティング >>

テニスボールが主人公の一風変わったシューティングゲームです。
見た目のシュールさとは裏腹に難易度は高めかも。
シューティングに慣れていないと全3面のクリアは難しいかもしれません。

< 攻略のヒント >

* 2体目のボスの2番目のパターンは、
  ボスの周りを回転するようにして避けます

< 工夫した点 >

実装に関しては、敵キャラの動きなどの複雑な処理を、メインループから呼ば
れるようなメソッドに細切れに記述するのが嫌だったので、スレッドを使って
コルーチン(もどき)を実装し、それを用いて実装したという点があります。し
かし、「継続」を使ってコルーチンを実装するのに比べて、スレッドを使って
実装する場合には、明示的にスレッドを終了しなくてはならず、面倒でした。

< 感想 >

今回自分でシューティングゲームを作ってみて、
世の中のシューティングゲームの敵キャラの動作や弾幕が、
いかに複雑なものだったのかを思い知りました。
このゲームも、もっともっと作りこみたかったのですが、
時間と私の気力の限界から、このようなものになりました。
特に、ラスボスの出来には不満が残ります。
